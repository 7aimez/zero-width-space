## What is ZWSP?

The _Zero Width Space_ is a unicode character with no width.

## How do you use ZWSP?

The HTML entity for ZWSP is `&ZeroWidthSpace;` or `&#8203;`. It is unicode character `U+200B`. The ZWSP is a similar feature to the HTML element `<wbr>` or `<wbr />`.
